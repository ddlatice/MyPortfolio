Crystal Clear UI

Why I created this project

I created this project serve as a simple baseline for websites I create in the future. Previous websites I have designed started off as intricate, but I am learning to simplify my workflow. I believe starting off with a minimalistic design and customizing futher is the best way for me construct unique projects.

How I created this project

I created this project with HTML, CSS, and JavaScript.

Challenges

I wanted to implement CSS Grid to display the shop item images, but I am still in the process of learning CSS Grid.

How I want to improve this project

I would like to add more JavaScript to this project and implement a fully functioning shopping basket. I would also like to be able to display the images with CSS Grid.
